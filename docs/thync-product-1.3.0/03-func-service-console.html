1. 개요

1.1 제품 및 대상 사용자

thynC는 SEERS사(씨어스테크놀로지)의 병원 입원환자용 무선 웨어러블 생체신호 실시간 모니터링 솔루션이며, 본 문서가 다루는 서비스 콘솔(thync.service)은 그중 의료진용 중앙 모니터링 웹 콘솔입니다. 병동의 간호사·의사·병원관리자가 환자 병상 배정, 실시간 중앙 모니터링, 부정맥·낙상 알람 대응, 측정이력·보고서 열람, 관리자 설정을 수행합니다.

항목내용
대상 사용자간호사(NURSE) · 의사(DOCTOR) · 병원관리자(MANAGER)
제품 모델thynC IP2140 (품목허가 제인 23-5092호)
사용자 매뉴얼ST-M6200-UM-KR Rev.6 (2025-11-26, 54p)
화면 표기 버전M1.3.0 · ⓒ 2025 seerstech

1.2 기술 개요

구분내용
아키텍처프레임워크 없는 Vanilla JS SPA (자체 "seers frontframework for vanilla"). 빌드/번들링 없음 — ES 모듈 dynamic import() + 자체 로더
라우팅History API SPA. index.html → js/index-init.js → js/common/loader.js가 URL /{controller}/{action}/{id}를 파싱해 js/controller/{controller}.js 동적 로딩
화면 템플릿Handlebars 4.7.7 문자열 (template/{테마}/content/*.js). 활성 테마 GBL.DESIGN.THEME = "thync"
주요 라이브러리jQuery 3.5.1 / jQuery UI 1.12.1, Bootstrap 3.3.2, D3 · Chart.js(파형·트렌드), SockJS 1.5.0 + STOMP 1.7.1(실시간), exceljs(엑셀), fili(필터), asmCrypto, coloris, Web Worker(차트 데이터 가공)
API 베이스{host}/mobiCAREConsole/API — 운영 https://www.api.seersthync.com, 온프레미스는 같은 호스트 :8080(http)/:8443(https) 자동 판별
배포정적 파일 서빙. deploy-server.sh가 서버에서 git fetch + git reset --hard. 캐시 무효화는 ?V=버전 쿼리스트링
